## Featured Projects

### Nonogram Solver

A command-line C++ application that loads and solves Nonogram puzzles.

Key features:

- Constraint satisfaction problem approach
- Domain propagation
- AC-3-style processing queue
- DFS backtracking
- Most-constrained-cell heuristic
- File-based puzzle input

[View Repository](https://github.com/thewisking/NonogramSolver)

### Pong Physics Engine

A two-player Pong game developed in C++ using a custom framework.

My contributions included:

- Time-of-impact collision detection
- Collision response
- Ball and paddle interactions
- Manual vector mathematics
- Multiple collision handling
- JSON configuration
- Debug and FPS overlays

The full source code is not publicly available because the project uses a teammate's private framework.
